form요소

웹브라우저를 이용해 사용자가 제어할 수 있는 ui

  • 전송할 데이터의 범위
  • 정규식 검사를 대체해주는 기능이 있다. (input에 required를 넣으면 된다.)
  • action 속성 : 데이터를 전송할 주소
  • method 속성 : get방식과 post방식이 있다.
    • get : 주소창을 이용해 텍스트로 데이터를 전송
    • post : 보안이 필요한 데이터, 바이너리파일을 전송

form의 구획요소

  • fieldset : 관련된 데이터의 범위를 지정할 때 fieldset 태그를 사용해준다.
  • legend : fieldset의 제목을 지정할 때 사용하는 태그이다.

form의 접근성

  • label 태그 : 접근성을 위해 label 태그를 사용하고 for= 속성을 이용하여 input의 id속성으로 연결시켜주어야한다.
  • title 속성 : label텍스트를 대체할 수 있는 속성

입력양식의 종류

  • input
  • select,option
  • textarea
  • button : form 안에서 사용되면 전송한다는 기능을 갖게된다.(새로고침이된다)
